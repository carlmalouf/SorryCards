# Sorry! Card Game

A Streamlit application that replicates the card-drawing experience from the classic board game Sorry!

## Features

- Authentic deck composition (45 cards matching the physical game)
- Random shuffle with proper card distribution
- Player turn tracking
- Card history display (last 3-5 cards)
- Card images for visual representation

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```

## Card Distribution

The deck contains 45 cards:
- 5x "1" cards
- 4x each of: Sorry!, 2, 3, 4, 5, 7, 8, 10, 11, 12
